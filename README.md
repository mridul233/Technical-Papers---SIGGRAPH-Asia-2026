# Scene-Lifted Dynamic Gaussian Splatting (SL-DGS)

This project presents a depth-free dynamic scene reconstruction framework using Gaussian Splatting and neural rendering.

## Features
- Depth-free reconstruction
- Monocular image processing
- Gaussian primitive optimization
- Real-time neural rendering

## Files
- `Hossain.ipynb` : Main training notebook

## Requirements
- Python
- PyTorch
- OpenCV
- NumPy

## Output
The model reconstructs dynamic scenes using Gaussian-based rendering with improved SSIM and LPIPS performance.
